# Face Detection System using OpenCV Python

This project implements a real-time face detection system using Python and OpenCV. The application uses a pre-trained Haar Cascade classifier to detect human faces in both images and live webcam video streams. The system performs image preprocessing techniques such as grayscale conversion and Gaussian blur to improve detection accuracy and reduce noise.

# Technologies Used
Python,
OpenCV,
Haar Cascade Classifier,
Computer Vision Techniques
